# 📦 CyberMind Installation Guide

## Super Simple - 3 Steps, 30 Seconds!

---

## Step 1: Download

Click the green **"Code"** button → **"Download ZIP"**

Extract the ZIP file anywhere on your computer.

---

## Step 2: Install in Chrome

1. Open Chrome
2. Type in address bar: `chrome://extensions/`
3. Turn ON **"Developer mode"** (top-right toggle)
4. Click **"Load unpacked"**
5. Select the **`extension`** folder
6. Done!

---

## Step 3: Use It!

- Browse any website
- CyberMind automatically scans it
- Click the extension icon to see results
- Badge shows threat level (green/yellow/red)

**That's it! You're protected!** 🛡️

## Troubleshooting

**Extension not loading?**
- Make sure you selected the `extension` folder, not the parent folder
- Check that `manifest.json` exists in the selected folder

**Not seeing the icon?**
- Click the puzzle icon in Chrome toolbar
- Pin CyberMind to toolbar

**Badge not changing?**
- Visit a website and wait 1-2 seconds
- Click the extension icon to see results
