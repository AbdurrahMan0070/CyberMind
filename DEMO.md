## 📋 Demo Script

### **Step 1: Show the Extension (10 seconds)**
1. Open Chrome
2. Point to CyberMind icon in toolbar
3. Say: *"CyberMind is an AI-powered security extension that automatically scans every website you visit"*

---

### **Step 2: Visit a Trusted Site (15 seconds)**
1. Go to **google.com**
2. Watch the badge turn **GREEN** ✓
3. Click the extension icon
4. Show:
   - Domain name displayed
   - "CLEAN" badge
   - Stats updating
5. Say: *"Google is recognized as a trusted domain - instant analysis, no waiting"*

---

### **Step 3: Visit an Unknown Site (15 seconds)**
1. Go to a random website (e.g., **example.com**)
2. Watch the badge change color
3. Click the extension icon
4. Show the threat assessment
5. Say: *"Unknown domains are analyzed in real-time and assigned risk levels"*

---

### **Step 4: Show Multilingual Support (20 seconds)**
1. Right-click extension icon → Inspect popup
2. Open Console
3. Type: `chrome.i18n.getUILanguage()`
4. Show current language
5. Say: *"CyberMind supports 7 languages - English, Spanish, French, German, Hindi, Chinese, and Japanese"*

---

### **Step 5: Show Scan History (15 seconds)**
1. Click extension icon
2. Point to stats:
   - Total scanned
   - Warnings
   - Threats
3. Say: *"All scans are stored locally - complete privacy, no data sent to servers"*

---

### **Step 6: Show Automatic Protection (20 seconds)**
1. Open multiple tabs quickly:
   - YouTube
   - Facebook
   - GitHub
   - Reddit
2. Show badges updating automatically
3. Say: *"Zero-click protection - scans happen automatically in the background"*

---

### **Step 7: Highlight Key Features (25 seconds)**
Point to README and say:
- ✅ **100% Local** - No servers, no API keys, no setup
- ✅ **Multilingual** - 7 languages for global accessibility
- ✅ **Privacy-First** - No tracking, no data collection
- ✅ **Open Source** - Transparent and auditable
- ✅ **Works on 99% of websites** - Including all major platforms

## 🎤 Q&A Preparation

### **Expected Questions:**

**Q: "How does it detect threats?"**
A: "It uses a curated database of trusted domains and analyzes domain patterns. Trusted sites like Google and Facebook are instantly recognized. Unknown sites are assessed based on domain characteristics."

**Q: "Why not use an API for threat intelligence?"**
A: "Privacy and speed. By working locally, we ensure zero data collection and instant results without network latency. Users don't need API keys or accounts."

**Q: "How do you handle false positives?"**
A: "We use conservative threat levels. Most sites are marked 'clean' or 'low'. Only suspicious patterns trigger warnings. Users can still access all sites - we inform, not block."

**Q: "What about updates to the threat database?"**
A: "The trusted domains list is built into the extension. For production, we'd implement automatic updates while maintaining local processing."

**Q: "Why 7 languages specifically?"**
A: "We chose the most widely spoken languages to maximize global impact - covering over 3 billion people worldwide."
