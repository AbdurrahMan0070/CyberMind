# 🚀 CyberMind Installation Guide

## For Judges & Reviewers

This guide will help you install and test CyberMind in **30 seconds**.

---

## ✅ What You Need

- **Google Chrome** or **Microsoft Edge** browser
- **That's it!** No Node.js, no npm, no terminal commands needed!

---

## 📥 Installation Steps

### Step 1: Download
- Go to: https://github.com/AbdurrahMan0070/CyberMind/releases/latest
- Download **`CyberMind-Complete.zip`**

### Step 2: Extract
- Right-click the downloaded ZIP file
- Select **"Extract All..."** or **"Extract Here"**
- You'll get a folder named **`extension`**

### Step 3: Install in Chrome
1. Open Chrome browser
2. Type `chrome://extensions/` in the address bar and press Enter
3. Turn ON **"Developer mode"** (toggle switch in top-right corner)
4. Click **"Load unpacked"** button
5. Select the **`extension`** folder you extracted
6. Done! ✅

---

## 🎯 How to Use

### Automatic Scanning
- Just browse normally!
- CyberMind automatically scans every website you visit
- Look at the extension icon in your toolbar:
  - 🟢 **Green checkmark** = Safe site
  - 🟡 **Yellow exclamation** = Warning
  - 🔴 **Red X** = Threat detected

### View Details
1. Click the **CyberMind icon** in your Chrome toolbar
2. See current page status and statistics
3. Click **"📊 VIEW DASHBOARD"** to see all scans

### Dashboard Features
- **Real-time stats**: Total scans, clean sites, warnings, threats
- **Scan history**: See all websites you've visited
- **Threat scores**: 0-100 risk assessment for each site
- **Auto-refresh**: Updates every 5 seconds

## ❓ Troubleshooting

### Extension not showing up?
- Make sure "Developer mode" is ON
- Try reloading the extension (click refresh icon)
- Restart Chrome

### Dashboard not opening?
- Make sure you clicked "Load unpacked" on the **extension** folder
- Check that all files are present in the extension folder
- Try removing and re-adding the extension

### Scans not appearing?
- Visit a few websites first
- Wait 1-2 seconds for scan to complete
- Click refresh button in dashboard
